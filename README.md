# Klassificering av ansiktsuttryck med CNN

## Beskrivning
Deep learning-projekt för att klassificera ansiktsuttryck från bilder 
med hjälp av ett Convolutional Neural Network (CNN).

## Dataset
FER-2013 — 35 887 gråskalebilder (48x48 pixlar) uppdelade i 7 klasser:
angry, disgust, fear, happy, neutral, sad, surprise

## Modell
CNN med tre Conv2D-block, Dropout och EarlyStopping.

## Köra projektet
pip install -r requirements.txt
Python 3.13
